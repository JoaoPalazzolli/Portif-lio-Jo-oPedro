Projeto DIFM - 04/2023 até o momento - 11 meses

Desenvolvedor Back-End

Fiquei responsável por fazer o backend do projeto DIFM, ele é uma aplicação web que tem como objetivo auxiliar na organização e controle de produtos em um sistema. Com essa aplicação, os usuários têm a capacidade de cadastrar, visualizar, atualizar e excluir produtos de um inventário.

Este projeto esta na versão 1 e é uma API REST criada com Spring framework, o banco de dados utilizado foi o MySQL que está sendo aberto através de um container no Docker e a autenticação foi criada com Spring Security e JWT.
